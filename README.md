# DevSports - Sistema de Agendamento de Espaços Esportivos

[cite_start]O **DevSports** é uma solução tecnológica desenvolvida como parte do Projeto de Extensão Interprofissional (Desenvolvimento Humano e Carreira) da **UNEX Itabuna**[cite: 2, 9]. [cite_start]O sistema visa resolver a falta de organização e os conflitos no uso de quadras esportivas públicas na cidade de Itabuna-BA[cite: 12, 42].

[cite_start]Através de uma plataforma digital, moradores podem visualizar horários disponíveis e realizar reservas antecipadas, promovendo o uso democrático e a preservação do patrimônio público[cite: 22, 23, 43].

## 🏗️ Estrutura do Sistema

[cite_start]O sistema é composto pelos seguintes módulos principais[cite: 48]:

* [cite_start]**Login & Cadastro:** Acesso seguro com coleta de nome, data de nascimento e telefone[cite: 49, 50].
* [cite_start]**Recuperação de Senha:** Validação via número de telefone para criação de nova credencial[cite: 51].
* [cite_start]**Agendamento (Formulário):** Seleção de modalidade (basquete, futsal, areia), local (ex: Beira Rio, Santo Antônio) e escolha de data/horário via calendário[cite: 52, 53, 54, 55].
* [cite_start]**Painel Administrativo:** Gestão centralizada de agendamentos, controle de presença (não comparecidos) e aplicação de restrições a usuários[cite: 57, 58, 59].
* [cite_start]**Relatórios:** Geração de documentos em PDF ou Excel com filtros de datas e tipos de dados[cite: 60, 61, 62].
* [cite_start]**Auditoria & Suporte (Opcional):** Monitoramento de logs suspeitos e canal de comunicação para problemas na plataforma ou nos espaços[cite: 63, 64, 65].

## 🌍 Impacto Social (ODS)

[cite_start]O projeto está alinhado aos Objetivos de Desenvolvimento Sustentável da ONU[cite: 12]:
* **ODS 3 (Saúde e Bem-Estar):** Incentivo à prática de atividades físicas e estilo de vida saudável.
* **ODS 11 (Cidades Sustentáveis):** Promoção de cidades mais inclusivas e uso adequado dos espaços públicos.
* **ODS 16 (Paz e Justiça):** Redução de conflitos sociais através de transparência e regras claras de utilização.

## 🚀 Tecnologias Utilizadas

O projeto utiliza as seguintes tecnologias para desenvolvimento e deploy:

* **Frontend:** HTML, CSS, Bootstrap e JavaScript.
* **Backend:** Python com o framework Flask.
* **Banco de Dados:** PostgreSQL (hospedado no Supabase).
* **Deploy/Servidor:** Render.

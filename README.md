# DevSports - Sistema de Agendamento de Espaços Esportivos

O **DevSports** é uma solução tecnológica desenvolvida como parte do Projeto de Extensão Interprofissional (Desenvolvimento Humano e Carreira) da **UNEX Itabuna**. O sistema visa resolver a falta de organização e os conflitos no uso de quadras esportivas públicas na cidade de Itabuna-BA.

Através de uma plataforma digital, moradores podem visualizar horários disponíveis e realizar reservas antecipadas, promovendo o uso democrático e a preservação do patrimônio público

## 🏗️ Estrutura do Sistema

O sistema é composto pelos seguintes módulos principais:

**Login & Cadastro:** Acesso seguro com coleta de nome, data de nascimento e telefone.
**Recuperação de Senha:** Validação via número de telefone para criação de nova credencial.
**Agendamento (Formulário):** Seleção de modalidade (basquete, futsal, volei), local (ex: Beira Rio, Santo Antônio) e escolha de data/horário via calendário.
**Painel Administrativo:** Gestão centralizada de agendamentos, controle de presença (não comparecidos).
**Relatórios:** Geração de documentos em PDF ou Excel com filtros de datas e tipos de dados.
**Auditoria (Opcional):** Monitoramento de logs suspeitos e aplicação de restrições a usuários.

## 🌍 Impacto Social (ODS)

O projeto está alinhado aos Objetivos de Desenvolvimento Sustentável da ONU:
* **ODS 3 (Saúde e Bem-Estar):** Incentivo à prática de atividades físicas e estilo de vida saudável.
* **ODS 11 (Cidades Sustentáveis):** Promoção de cidades mais inclusivas e uso adequado dos espaços públicos.
* **ODS 16 (Paz e Justiça):** Redução de conflitos sociais através de transparência e regras claras de utilização.

## 🚀 Tecnologias Utilizadas

O projeto utiliza as seguintes tecnologias para desenvolvimento e deploy:

* **Frontend:** HTML, CSS, Bootstrap e JavaScript.
* **Backend:** Python com o framework Flask.
* **Banco de Dados:** PostgreSQL (hospedado no Supabase).
* **Deploy/Servidor:** Render.
